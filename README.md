# snow-renamer

Auto rename all images with AI Image Captioning

# Models

This project used coco-ssd by tensorflow

# Requirements

1. Node.js v16
2. yarn

# Installation

1. Clone this project and `cd` into it
2. `yarn install`

# Usage

`node index.js path-to-your-image-folder`
